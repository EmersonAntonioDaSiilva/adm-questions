O jMagnaSolr tem por objetivo criar um cluster e "indexar" documentos a esse (cluster).

Para que seja realizado esse processo deve-se passar as credenciais corretas do Retrieve and Rank e do Document Conversion.

O Solr.zip contém o programa que deve ser descompactado na pasta raiz (C:\). Esse programa é indicado ser executado no Terminal através do comando

java -jar jMagnaSolr (trata-se de um Arquivo Jar Executável gerado pelo Export Runnable Jar File do Eclipse).
