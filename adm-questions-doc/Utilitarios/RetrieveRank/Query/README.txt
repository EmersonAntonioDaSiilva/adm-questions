O query-solr-magna tem por objetivo realizar pesquisa a partir de um cluster.

Para que seja realizado esse processo deve-se passar as credenciais corretas. O Id Cluster pode ser obtido com o clique do botão ao lado da caixa de texto; mesma coisa com a collection.

java -jar query-solr-magna ou duplo clique para executar (trata-se de um Arquivo Jar Executável gerado pelo Export Runnable Jar File do Eclipse).
